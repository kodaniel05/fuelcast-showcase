# Images

This folder is reserved for public-facing visuals.

## Screenshot Checklist

Add screenshots to `images/screenshots/` using these filenames:

- `dashboard-home.png`
- `forecast-cards.png`
- `us-state-map.png`
- `state-rankings.png`
- `ai-analyst-takeaway.png`
- `model-metrics.png`
- `pipeline-health.png`
- `drivers-and-signals.png`
- `top-movers.png`

Before publishing, review each screenshot to make sure it does not expose API keys, private environment values, database connection strings, raw client data, internal file paths, or implementation details.

## Diagram Checklist

Add exported diagrams to `images/diagrams/` only if they are safe to publish. Mermaid diagrams are already included in the Markdown docs, so additional image diagrams are optional.
